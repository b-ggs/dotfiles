# dotfiles

Personal configurations for nvim, zsh, tmux, etc.

## TL;DR

```
git clone git@github.com:b-ggs/dotfiles.git ~/.files
~/.files/common/install
~/.files/zsh/install
~/.files/vim/bin/install_nvim
~/.files/vim/bin/install_vim
~/.files/git/install
~/.files/tmux/install
~/.files/tmuxinator/install
```
