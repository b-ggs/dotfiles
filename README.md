# dotfiles ![travis][travis]

Personal configurations for nvim, zsh, tmux, etc.

## TL;DR

```
git clone git@github.com:b-ggs/dotfiles.git $HOME/.files

make \
  brew-common \
  brew-macos \
  brew-linux \
  nvim-symlink \
  nvim-configure \
  zsh-symlink \
  zsh-configure \
  tmux-symlink \
  git-configure \
  git-symlink \
  um-symlink
```

[travis]: https://travis-ci.org/b-ggs/dotfiles.svg?branch=master
